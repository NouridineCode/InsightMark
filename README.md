# InsightMark
Plateforme d'analyse des données clients développée avec Python pour aider les entreprises à comprendre leurs clients, anticiper les comportements et améliorer leurs décisions.

#######
InsightMark is an AI-powered customer analytics platform developed in Python that helps businesses analyze customer behavior, predict churn, forecast sales, detect anomalies, and generate actionable business insights through interactive dashboards and intelligent analytics.

## Fonctionnalités
-  Tableau de bord interactif
-  Segmentation des clients
-  Analyse RFM (Récence, Fréquence, Montant)
-  Prédiction du risque de perte de clients (Churn)
-  Prévision des ventes
-  Détection d'anomalies
-  Génération d'insights grâce à l'intelligence artificielle
-  Importation et exportation de données
-  Génération de rapports
-  Statistiques commerciales
-  Simulations et aide à la décision
-  Analyse du comportement des clients

##  Technologies utilisée
- Python 3
- Tkinter
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- SQLite


## Installation

### 1. Cloner le dépôt

```bash
git clone https://github.com/NouridineCode/InsightMark.git
```

### 2. Accéder au dossier du projet

```bash
cd InsightMark
```

### 3. Créer un environnement virtuel

```bash
python -m venv .venv
```

### 4. Activer l'environnement

**Sous Windows**

```bash
.venv\Scripts\activate
```

**Sous Linux / macOS**

```bash
source .venv/bin/activate
```

### 5. Installer les dépendances

```bash
pip install -r requirements.txt
```

### 6. Lancer l'application

```bash
python app.py
```

## Objectifs
InsightMark permet aux entreprises de :

- Comprendre le comportement de leurs clients.
- Identifier les clients les plus fidèles.
- Détecter les clients susceptibles de partir.
- Prévoir les ventes futures.
- Détecter des anomalies dans les données.
- Optimiser les stratégies marketing.
- Faciliter la prise de décision grâce à des analyses intelligentes.
